Objective

Our project aims to enhance safety in sewer and septic tank cleaning operations to reduce risks and saving lives of manual scavengers during these crucial tasks using Internet of Things (IoT).

Features

1. MQ135 sensor for detecting hazardous gases

2. DHT11 sensor for temperature and humidity monitoring

3. PIR sensor for continuous motion detection

4. LCD display on the device for showing real time environmental data

5. LED lights and buzzer for various alerts

6. NodeMCU ESP8266 module for WiFi connectivity

7. Web app dashboard for monitoring the condition inside
